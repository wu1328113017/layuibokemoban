安装与搭建以上都有，所以就不写了，主要写一下可能会发生的问题

1.文章上的关闭防火墙的命令可能没用，需要使用systemctl stop firewalld

或service  iptables stop命令，可以借鉴https://blog.csdn.net/qq_35971258/article/details/79318842

2.关闭防火墙的作用是取消外界访问的限制，即不会拦截别的电脑访问服务器的请求

3.安装mysql服务成功了后要登录的时候如果先输入用户名再输入密码可能会输入不了密码，就是可能用户名已经输入好了并回车，再去输入密码是输入不了的，所以要一次性直接用户名密码一起输入

借鉴https://www.jianshu.com/p/3e399c76b132

4.当输入命令 scripts/mysql_install_db --user=mysql  时可能会报ERROR: please install the following Perl modules before executing ./scripts/mysql_install_db:Data::Dumper的错误，解决办法是下载autoconf

即输入命令 yum -y install autoconf

5.当输入命令 ./mysql_install_db --verbose --user=root --defaults-file=/etc/my.cnf --datadir=/usr/local/mysql/data --basedir=/usr/local/mysql --pid-file=/usr/local/mysql/data/mysql.pid --tmpdir=/tmp时可能会报error while loading shared libraries: libaio.so.1: cannot open shared object file: No such file or directory这个问题，解决办法：安装libaio-devel，即输入命令yum -y install libaio-devel

因为是centos系统，所以要额外的下载一个numactl，输入命令：yum -y install numactl

借鉴https://ask.csdn.net/questions/682718
