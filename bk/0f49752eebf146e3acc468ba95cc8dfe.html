完成jdk的搭建后，接下来就是进行浩浩荡荡的Java开发了，我们可以随便打开一个文件目录或直接在桌面上新建一个txt文件，并改后缀为java，改后缀就是文件名后面一个”.“之后的英文，例如我们先将这个txt文件取名为HelloJava，那么这个txt文件的完整文件名是HelloJava.txt，这个txt就是后缀名，我们要将这个后缀名改成java，即将HelloJava.txt改成HelloJava.java，可能会提醒改变后缀名会带来一些什么影响，不用管它，直接确认或是继续，然后打开文件编辑，此时双击是打开不了或编译不了或者是给你一个打开方式的选择，这时选择记事本的打开方式，如果打开不了就右击然后选择打开方式的选项，选择记事本打开，然后要改变文件的编码，这个自己去百度怎么改变记事本的编码，因为记事本种类不同，根据自己的记事本去改变编码，要改成UTF-8，编码是国际的一个规范，就像国家与国家之间有语言之分一样，可以自行百度，在记事本上写Java代码，以后我们都是称编写Java语言的这些个语言为代码，

        写入：

        public class HelloJava{# ”#“为注释，注意Java文件名要大写，而且这个class后面的英文要和文件名一样，这个

#我们是叫类名，而且每一个代码之间都要空格，比如public和static就要空格隔开，

                 public static void main(String[] args){

                        System.out.println("hello！Java”);

                 } 

        }

       保存，然后打开命令符窗口，打开方式为同时按下window键(就左下角那个一个窗口符号的键)和r键，按下后会出现一个框，里面还有一个可以输入的框，在这个可输入的框中输入  cmd，然后回车，之后就会出现一个窗口，背景黑乎乎的，这就是命令符窗口，在里面输入cd：你放置Java文件的所在路径，然后再输入javac HelloJava.java，回车后系统执行这段命令之后会在你java文件所在目录生成一个后缀名为class的文件，我们习惯称之为“点class文件”，在Java里面这是叫类文件，是Java文件经过编译后产生的文件，然后再在命令符窗口里面输入java HelloJava，注意，这里不需要写上后缀名，即不要写成java HelloJava.class，然后回车，就会看到命令符窗口输出一段话：

                          hello! Java

        此刻，我们的Java第一个程序已成功编写成功，不过这只是跨入Java语言的第一步，后面还有好多东西，而且我们后面都不是这种直接在目录先创建文件然后输入命令符操作的风格，而是下载专门的Java开发工具来做这些东西，而这些开发工具都是可以在网上下载的，目前比较流行的有idea和eclipse，个人推荐初学者下载eclipse更好一些
